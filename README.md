# 🏆 철도안전법 기말고사 족보 퀴즈 & 학습 웹 앱

이 프로젝트는 철도안전법 기말고사(약 25문제)를 앞두고 교수님이 대놓고 짚어주신 **[기말고사 25문제 정답 족보]**를 바탕으로 제작한 모바일 친화형 웹 애플리케이션입니다. 노베이스 상태에서도 기말고사에서 좋은 성적을 거둘 수 있도록 속성 과외(학습)와 퀴즈(시험) 기능을 통합 제공합니다.

---

## ✨ 주요 특징 및 기능

1. **리퀴드 글래스 (Liquid Glass) UI 디자인**
   - 차분한 미드나잇 블루, 딥 바이올렛, 시안 톤의 **유기적으로 넘실거리는 플로팅 블롭(Floating Blobs)** 배경 애니메이션.
   - 강력한 흐림 효과(`backdrop-filter: blur(25px)`)가 적용된 반투명 프로스트 글래스 카드 렌더링.
   - 모바일 세로 뷰포트(`100dvh`)에 최적화되어, 바디 전체 스크롤 없이 컴팩트한 앱처럼 동작합니다.

2. **노베이스 속성과외 (학습 모드)**
   - 족보의 6개 구역에 포함된 복잡한 철도 규정 및 용어(입환, 3NC, 준사고/운행장애 등)를 비전공자 관점에서 풀어서 쉽게 설명합니다.
   - 모바일 네이티브 제스처인 **가로 스와이프(수평 스크롤 스냅)**로 좌우 카드를 넘겨가며 편하게 읽을 수 있습니다.

3. **3단계 난이도 퀴즈**
   - **🌱 순한맛 (쉬움)**: 제한 시간 없이 문제를 풀며, 문제 풀이 도중 실시간으로 **[💡 족보 힌트 보기]**를 열고 닫을 수 있습니다.
   - **🔥 보통맛 (보통)**: 제한 시간이 없으며, 보기를 선택하여 문제를 푼 직후 하단 해설 시트를 통해 교수님 팁을 제공받습니다.
   - **🌶️ 매운맛 (교수님 복수 모드)**: **문항당 15초의 타임어택** 타이머가 작동하며 시간 초과 시 오답 처리됩니다. 매운맛 전용으로 **지문과 보기를 까다롭게 꼰 고난도 퀴즈 셋**이 출제됩니다.

4. **재수강 등급 & 오답 노트**
   - 시험 종료 후 70점 미만의 낮은 점수를 기록할 경우, 무서운 F 학점 대신 유머러스하고 친숙한 **'재수강'** 등급 배지와 잔인한(?) 코멘트가 나타납니다.
   - 틀린 문제가 있을 경우, 해당 난이도 설정 그대로 **[오답만 다시 풀기]** 기능을 통해 복습할 수 있습니다.
   - 언제 어디서나 우측 상단의 📖 아이콘을 눌러 간편 족보 요약본을 슬라이드로 꺼내볼 수 있습니다.

## 📂 파일 구성

- **[index.html](index.html)**: 로비, 학습, 퀴즈, 결과 화면과 하단 해설 시트가 하나로 구성된 단일 페이지 웹 구조 (SPA).
- **[style.css](style.css)**: 리퀴드 블롭 애니메이션 및 모바일 가로 스크롤 스냅, 글래스모피즘 전용 스타일시트.
- **[questions.js](questions.js)**: 6개 구역 학습 데이터 및 난이도별(순한맛/매운맛) 27문항 퀴즈 데이터베이스.
- **[app.js](app.js)**: 스와이프 동작 감지, 타이머 작동, 정/오답 판정 및 난이도 분기 처리를 총괄하는 스크립트.
