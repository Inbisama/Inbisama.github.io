GitHub 블로그 인증

이 블로그는 사용자의 브라우저에서 GitHub API를 통해 직접 글을 쓰고 업로드합니다. 본인 인증을 위해 GitHub Personal Access Token (PAT)이 필요합니다.

체크 해제 시 브라우저 탭을 닫으면 로그인 정보가 안전하게 자동 파기됩니다.

💡 GitHub 토큰 발급 가이드

  1. GitHub 로그인 후 Settings > Developer Settings > Personal Access Tokens > Fine-grained tokens로 이동합니다.
  2. Generate new token을 클릭합니다.
  3. Repository access에서 Only select repositories를 선택하고 본인의 Inbisama.github.io 저장소를 지정합니다.
  4. Permissions에서 Repository permissions > Contents 권한을 Access: Read and write로 설정합니다.
  5. 토큰을 생성하여 복사한 뒤 위 칸에 입력합니다.
